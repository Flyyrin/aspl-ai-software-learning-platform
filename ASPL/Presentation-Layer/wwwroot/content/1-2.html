Variables and Data Types

In this chapter, we'll delve deeper into variables and data types in Python.

Variables

Variables are used to store data values in Python. Unlike some other programming languages, Python does not require declaring variables with a specific data type. The type of the variable is inferred at runtime based on the value assigned to it.

Variable Naming Rules

When naming variables, remember the following rules:

  • A variable name must start with a letter or the underscore character (_).
  • A variable name cannot start with a number.
  • A variable name can only contain alphanumeric characters and underscores (A-z, 0-9, and _).
  • Variable names are case-sensitive.

Data Types

Python supports various data types to represent different types of data. Some common data types include:

  • Integer: Represents whole numbers (e.g., 5, -10).
  • Float: Represents decimal numbers (e.g., 3.14, -0.001).
  • String: Represents a sequence of characters (e.g., "hello", 'world').
  • List: Represents an ordered collection of items (e.g., [1, 2, 3]).
  • Tuple: Similar to lists, but immutable (e.g., (1, 2, 3)).
  • Dictionary: Represents a collection of key-value pairs (e.g., {'name': 'Alice', 'age': 30}).
  • And more...

Examples

Let's look at some examples of variables and their data types:


# Integer
x = 5

# Float
y = 3.14

# String
name = "Alice"

# List
numbers = [1, 2, 3, 4, 5]

# Dictionary
person = {'name': 'Bob', 'age': 25}

Type Conversion

You can convert between different data types using built-in functions. For example:


# Convert integer to float
x = 5
y = float(x)

# Convert float to integer
z = int(3.14)

# Convert integer to string
s = str(10)

Conclusion

Variables and data types are fundamental concepts in Python programming. Understanding how to work with variables and manipulate data types is essential for building more complex programs. In the next chapter, we'll explore operators and expressions in Python.
